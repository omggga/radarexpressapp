# Radar Price Tool

# start
npm run start

# stop
npm run stop

# build new html
npm run build


# Install:
- install nodejs
- install pm2
- clone git repo
- put outside folder ecosystem.config.js with env variables: NODE_ENV, PORT, PG_CONNECT
- npm install
- build project
- start project
